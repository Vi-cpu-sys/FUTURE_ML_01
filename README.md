# 📈 Sales & Demand Forecasting for Businesses

## 📌 Project Overview

This project was developed as part of the **Future Interns Machine Learning Internship - Task 1**.

The objective is to build a Machine Learning model that forecasts future sales using historical business data. The project includes data preprocessing, feature engineering, exploratory data analysis, model training, evaluation, and business insights.

---

## 📂 Dataset

- Historical Sales Dataset
- Features include:
  - Order Date
  - Sales
  - Category
  - Region
  - Quantity
  - Profit

---

- Python
- Pandas

- Scikit-learn
- Jupyter Notebook

## 🚀 Project Workflow

1. Data Loading
2. Data Cleaning
3. Time-based Feature Engineering
4. Exploratory Data Analysis
5. Train-Test Split
6. Random Forest Regression Model
7. Model Evaluation
8. Sales Prediction
9. Business Insights

## 📊 Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
